## 决策记录

> 记录开发过程中遇到的决策性问题及其结论，避免重复讨论。

### 2026-05-29

**Q: stop-the-world 暂停机制怎么实现？**

A: 直接在 `on_step` 中调用 LLM，不另开线程。LLM 调用期间 `on_step` 不返回，burnysc2 游戏循环自然阻塞，游戏自动暂停。

**Q: LLM 对接方式？**

A: 复用 `E:\Code\python\scientific research\SC2\agent\LLMClient.py` 中的 `UnifiedLLMClient` 及其多供应商路由、重试、Token 预算机制。通过 `llm_clients.json` 配置供应商（DeepSeek + Qwen），使用 OpenAI compatible SDK 进行 function calling。

**Q: 日志系统设计？**

A: 使用 loguru 替代 stdlib logging，JSON 格式输出到文件 + 可读格式到 stderr，每局一个 `storage/{game_id}/logs/` 目录。详见 `技术设计.md` 第 13 节。
