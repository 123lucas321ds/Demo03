# 审查知识

## 审查时的关注点

### 资源方面
- 命令总消耗是否超过当前 + 预期收入？
- 是否预留了足够资源给后续紧急操作？
- 气矿是否被忽略超过了 2 分钟？

### 生产方面
- 是否所有生产建筑都被分配了任务？
- 是否有建筑在空闲等待？
- SCV 是否持续训练？

### 补给方面
- 当前补给是否接近上限？
- 是否有 SupplyDepot 在建或已规划？

### 军事方面
- 是否有应对敌方可能的骚扰的单位？
- 兵力是否过于分散？

### 科技方面
- 当前科技进度是否符合战略方向？
- 前置建筑是否已建造？
