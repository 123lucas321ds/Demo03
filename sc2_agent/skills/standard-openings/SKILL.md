# 标准开局模板

## 使用方式

使用 plan.build_order 工具获取模板详情，不要凭记忆编造。

## 已支持模板

- `terran_1rax_expand` — 单兵营扩张开局（安全经济型）
- `terran_reaper_expand` — 死神扩张开局（侦察骚扰型）

Agent 应根据敌方种族和自身策略选择合适的模板，仍须使用 plan.build_time 和 plan.simulate 自行推算 at_time。
