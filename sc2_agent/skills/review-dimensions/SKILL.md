# 审查维度

review.logic Sub-Agent 从四个维度审查命令时间线：

## 1. 对抗合理性
- 是否考虑了已知敌方单位和可能的战术？
- 是否过度投入经济而忽视了防御？
- 是否对敌方可能的骚扰有应对措施？

## 2. 资源合理性
- 命令消耗是否在预测资源范围内？
- 是否合理分配了矿和气？
- 是否有资源过度堆积而未被利用？

## 3. 战略一致性
- 命令是否服务于当前 game_state.md 中记录的战略判断？
- 每个阶段的优先级是否被正确执行？
- 是否遵循了选定的开局模板？

## 4. 时间线合理性
- 命令的 at_time 是否按建造时间正确推算？
- 同一建筑是否有时间冲突的任务？
- 补给是否有被卡住的可能？
